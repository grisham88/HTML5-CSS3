CSS Selektoren: Adjacent Sibling Selektor

Syntax:
vorgaenger + nachfolger

Dieser zusammengesetzte Selektor erfasst ein Element nachfolger, wenn ihm ein Element vorgaenger direkt vorangeht (hierunter versteht man, dass der andere Container des Vorgängers unmittelbar vor dem Starttag des Nachfolgers geschlossen wird). Zwischen den Einzelselektoren steht das Kombinatorsymbol +, optional umgeben von Leerzeichen.

Beispiel:
/* erfasst <b> nach <a>: */
a + b

Kompatibilität:
Der Adjacent-Sibling-Selektor wird von allen modernen Browsern unterstützt (Mozilla, Firefox, Safari, Opera; auch IE5/Mac). Der Internet Explorer für Windows hingegen kennt den Child-Selektor erst ab Version 7.0.
