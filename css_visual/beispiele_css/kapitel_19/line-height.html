Textproperties: line-height

Das line-height-Property bestimmt die Zeilenhöhe innerhalb eines Containers und damit indirekt den Zeilenabstand. Wird eine Zeilenhöhe niedriger als die vorherrschende Schrifthöhe angegeben, so führt dies zu einer Überlagerung der Zeilen untereinander. Als Werte übergeben können Multiplikationsfaktoren, Längenmaße mit Einheit oder Prozentangaben, die sich auf den Grundlinienabstand der Zeilen in Abwesenheit des Properties bezieht.

Werte:
normal, inherit, [faktor], [laenge], [prozent]

Beispiel:
p { line-height: 0.9;} /* Faktor */
p { line-height: 1.2em;} /* Längenangabe */
p { line-height: 140%;} /* Prozentangabe */
