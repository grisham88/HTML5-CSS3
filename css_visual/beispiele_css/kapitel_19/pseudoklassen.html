CSS Selektoren: Pseudoklassen

Syntax:
typ-selektor:pseudoklasse
typ-selektor.klassenname:pseudoklasse

Die so genannten Pseudoklassen werden in Zusammenhang mit dem Stylen von Hyperlinks benötigt (theoretisch sind einige davon jedoch auf alle Elemente anwendbar) – sie beziehen sich im Gegensatz zu „echten“ Klassen nicht auf das mit dem vorausgehenden Typ-Selektor bezeichnete Element sondern dessen Zustand und werden daher (mit Ausnahme von :lang) auch oft auch als „dynamische Klassen“ bezeichnet. Pseudoklassen können mit normalen Klassen kombiniert werden, wobei die Pseudoklasse am Ende stehen muss (s.o.).

Achtung bei Pseudoklassen und Links:
Beim Zuweisen mehrerer Pseudoklassen an Links ist die Definitionsreihenfolge maßgeblich. Folgende Reihenfolge der Deklarationen sollte im Stylesheet unbedingt eingehalten werden:

a:link { ... }
a:visited { ... }
a:hover { ... }
a:active { ... }

Will man gleichzeitig auch den Fokus-Zustand :focus erfassen, so muss dieser vor dem :active-Zustand definiert werden:

a:link { ... }
a:visited { ... }
a:hover { ... }
a:focus { ... }
a:active { ... }

:active

Die Pseudoklasse :active stylt den Zustand aktiver Elemente, also solche, die durch den User angeklickt werden (onmousedown). Das Element bleibt für die Dauer des Klicks aktiv. Die Pseudoklasse wird zwar meist für Links verwendet, ist jedoch für alle klickbaren (aktivierbaren) Elemente gestattet. Dies ermöglicht es (in der Theorie), auch Formularfeldern, Tabellenzellen oder <div>-Bereichen aktive Zustände zuzuweisen.

Beispiel:
a:active {color:red;}
a:link {text-decoration:none;}

:focus

Die Pseudoklasse :focus stylt den Fokus-Zustand von Elementen. Dies trifft auf Hyperlinks zu, die per Tabulator angewählt, aber noch nicht aktiviert sind, sowie auf Formulareingabefelder, die angeklickt wurden. Es kann jeweils nur ein einziges Element den Fokus besitzen.

Anmerkung:
Internet Explorer kann :focus nicht auf Formularelemente anwenden und unterscheidet bei Links nicht zwischen :focus und :active. In anderen Browsern kann ein per Tab angewählter Link (Fokuszustand) dagegen gezielt gestylt werden.

Beispiel:
input:focus { background-color:#DDDDDD; }

:hover

Die dynamische Pseudo-Klasse :hover bestimmt das Aussehen des Links (bzw. des gestylten Elements) im Zustand des MouseOvers. Prinzipiell kann :hover zwar auf alle Elemente angewendet werden, im Internet Explorer und älteren Opera-Browsern beschränkt sich der Einsatz jedoch auf Hyperlinks.

Beispiel:
a:link {text-decoration:none;}
a:hover {text-decoration:underline;}

:lang

Die Pseudoklasse :lang formatiert ein Element in Abhängigkeit der eingestellten Sprache. Diese muss durch das lang-Attribut des Elements angegeben sein.

Beispiel:
p { color:#999999; font-family:arial;}
p:lang(de) { color:#000000; }
p:lang(en) { color:#666666; font-weight:bold;}

:link

Die Pseudo-Klasse :link bestimmt die Darstellung nicht besuchter Hyperlinks, also die Ausgangspräsentation des Links.

Beispiel:
a:link {text-decoration:none;}

:visited

Die Pseudo-Klasse :visited bestimmt die Darstellung besuchter Hyperlinks. Auf diese Weise können besuchte von unbesuchten Links unterschieden werden, um dem Nutzer der Seite den Überblick zu erleichtern.

Beispiel:
a:visited {color:#666666;}
