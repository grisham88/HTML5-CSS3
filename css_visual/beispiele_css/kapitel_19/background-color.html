Farben und Hintergründe: background-color

Das Property background-color bestimmt die Hintergrundfarbe des Containerinhaltsbereichs eines Elements. Handelt es sich um den <body>-Container, ist dies die Hintergrundfarbe des gesamten Dokuments, die an die in diesem enthaltenen Container weitervererbt wird.

Werte:
[farbname], [farbwert], transparent, inherit

Beispiel:
/* abgekürzte Farbwertsyntax: */
body { background-color: #F00 }
body { background-color: transparent; }
body { background-color: #dddddd; }
