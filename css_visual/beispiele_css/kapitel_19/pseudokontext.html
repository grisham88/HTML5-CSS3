CSS Selektoren: Pseudo-Kontextselektoren

Auch in Zusammenhang mit Kontextselektoren spricht man von einem „Pseudoselektor“, sofern dieser einen künstlich beschränkten Bereich erfasst.

:first-child

Spricht innerhalb der Dokumentstruktur ein Element an, wenn es das erste Kind seines Elternelements ist. In einer <ul>-Liste ist der erste <li>-Eintrag das erste Kind von <ul> und kann mit li:first-child anders formatiert werden.

Achtung:
Nicht das Elternelement ist hier ausschlaggebend, sondern lediglich, dass das Zielelement ein „erstes Kindelement“ ist, egal von welchem Elternelement.

Beispiel:
li:first-child { font-weight:bold; }

/* ist gleichbedeutend mit: */
* li:first-child { font-weight:bold; }

/* erste Zeile oder Spalte einer Tabelle: */
tr:first-child { background-color:#dddddd; }
td:first-child { background-color:#dddddd; }
