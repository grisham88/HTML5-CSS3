Listenstyle Properties: list-style

Das list-style-Property ist ein Sammelproperty für die drei grundlegenden Listenelementeigenschaften list-style-type, list-style-image und list-style-position, die hier einzeln oder gemeinsam zugeordnet werden können. Wird ein Einzelproperty nicht übergeben, so gilt dessen Defaultwert.

Anmerkung:
Der Listentyp list-style-type und die Listengrafik list-style-image können nicht gleichzeitig eingesetzt werden.

Werte:
[list-style-type]? [list-style-image]? [list-style-position]?

Beispiel:
li { list-style:url('pfeil.gif') inside; }
li { list-style:decimal-leading-zero; }
li { list-style:url('pfeil.gif') inside; }
