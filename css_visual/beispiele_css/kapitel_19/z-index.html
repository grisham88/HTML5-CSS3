Positioning-Properties: z-index

Das z-index-Property bestimmt die Tiefenstaffelung sich überlagernder, positionierter Elemente. Es können ganze Zahlen oder der Wert auto übergeben werden.

Werte:
[numerisch], auto, inherit

Ausprobieren:
z-index:

Sie können den z-index der roten Ebene beeinflussen:

z-index:15

z-index:20

z-index:25
