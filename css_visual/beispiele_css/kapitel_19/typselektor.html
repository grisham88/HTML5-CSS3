CSS Selektoren: Typ-Selektor

Syntax:
tagname

Der Typ-Selektor (Element-Selektor) zielt auf Elemente des genannten Tagnamens (generic identifier). Er besteht daher im einfachsten Fall aus einem Tagbezeichner (sowohl ein HTML-Tagbezeichner als auch gegebenenfalls ein XML-Tagbezeichner).

Beispiel:
p {font-family:arial; font-size:0.8em; }

Kompatibilität:
Der Typ-Selektor wird von allen Browsern unterstützt.
