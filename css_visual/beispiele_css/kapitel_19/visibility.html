Positioning-Properties: visibility

Das visibility-Property steuert die Sichtbarkeit eines Elementcontainers. Dieser nimmt, unabhängig von den dort vorgenommenen Einstellungen stets den gleichen Raum an der ihm angewiesenen Position ein.
Der Wert collapse wirkt auf Tabellenelemente wie Zeilen <tr> und Spalten <col> bzw. Spaltengruppen <colgroup> und entfernt diese aus der Anzeige. Hierbei wird der von ihnen sonst eingenommene Platz jedoch freigemacht, so dass im Flow folgende Elemente nachrücken. Auf Nicht-Tabellenelemente angewendet, wirkt die Einstellung collapse wie hidden.

Werte:
visible, hidden, collapse, inherit
