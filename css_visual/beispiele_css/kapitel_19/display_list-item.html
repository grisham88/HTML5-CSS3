Displayproperties: display:list-item

Das Element wird als Listenelement dargestellt. Ein übergeordneter Listencontainer muss hierfür nicht definiert werden. Ein so definiertes Listenelement ist grundsätzlich ein Blockelement, wobei die üblichen Defaultverhalten in Bezug auf Bullets gelten, nicht jedoch auf Margins.

Beispiel:
span { display:list-item; }
