CSS At-Rules: @page

Der Befehl @page betrifft die seitenorientierte Ausgabe des Dokuments, womit sowohl Druckausgabe als auch Projektion im Rahmen einer Präsentation abgedeckt werden soll. Bestimmt wird die Aufteilung des Dokuments in einzelne Seitenblöcke, wobei die Seitenabmessungen und -ränder bestimmt werden können.

Beispiel:
@page {
size: 21cm 29.7cm;
margin-left: 3cm;
margin-top: 4.5cm;
margin-bottom: 3cm;
}

Kompatibilität:
Derzeit wird die seitenorientierte Ausgabe mit @page in aktuellen Browsern nicht unterstützt.
