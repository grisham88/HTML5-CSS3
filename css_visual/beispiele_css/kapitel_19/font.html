Fontproperties: font

Das Sammelproperty font bietet die Möglichkeit, alle sechs zur Schriftpräsentation dienenden Properties gemeinsam festzulegen. Es sind, getrennt durch Leerzeichen, die Werte für die Subproperties font-style, font-variant, font-weight, font-size/line-height und font-family anzugeben. Soll eines der Subproperties nicht näher bestimmt werden, so kann an Stelle eines Wertes das Schlüsselwort inherit übergeben werden. Ein Wert für line-height ist stets zusammen mit font-size anzugeben, wobei zwischen beiden ein trennender Slash stehen muss: font-size/line-height.

Werte:
[font-style] [font-variant] [font-weight] [font-size(/line-height)?] [font-family]

Beispiel:
p {font: 12pt/14pt sans-serif}
p {font: 80% sans-serif}
p {font: x-large/110% "Times New Roman", serif}
