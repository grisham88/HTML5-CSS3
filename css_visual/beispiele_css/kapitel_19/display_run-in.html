Displayproperties: display:run-in

Das Element bricht nach dem vorangehenden Inhalt um, verhält sich jedoch gegenüber seinem Nachfolgerelements, entsprechend dessen Eigenschaften, unterschiedlich: Ist der Nachfolger ein Blockelement, so läuft das Run-in-Element als Inlineelement in dessen erste Zeile hinein. Handelt es sich um ein gefloatetes Element, so verhält das Run-in-Element sich hingegen wie ein Blockelement und bricht vor den Nachfolger um.. (Derzeit nur in Opera)

Beispiel:
span { display:run-in; }
