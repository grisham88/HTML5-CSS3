Textproperties: white-space

Das white-space-Property legt die Behandlung von Weißraumzeichen (Zeilenumbruch, Tabulator, Leerzeichen) im betreffenden Elementcontainer fest.

Werte:
normal, pre, nowrap, inherit

Beispiel:
pre { white-space: pre }
p { white-space: normal }
td[nowrap] { white-space: nowrap }
