Boxproperties: min-height

Das min-height-Property bestimmt die minimale Höhe, die der Innenbereich des Containers nicht unterschreiten darf. Es kann eine Längenangabe übergeben werden, oder ein Prozentwert, der sich auf die Höhe des Elternelements bezieht.

Werte:
[laenge], [prozent], inherit

Beispiel:
p { min-height:250px; }
p { min-height:60%; }
p { min-height:inherit; }
