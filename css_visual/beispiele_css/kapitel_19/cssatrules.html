CSS At-Rules

Die CSS "At-Rules" gelten entweder für das gesamte Stylesheet, um dort bestimmte Voreinstellungen zu treffen, für die keine Selektoren zuständig sind (@charset, @font-face), oder importieren externe Stylesheetregeln aus einer anderen CSS-Datei (@import). Drittens dienen sie zur Formulierung bedingter Abschnitte innerhalb des Styelsheets, die nur unter bestimmten Umständen verwendet werden sollen (@media in Abhängigkeit vom Medientyp des Client, @page beim Ausdruck des Dokuments). Die At-Rules besitzen daher eine gewisse Sonderstellung gegenüber den restlichen Ausprägungen der CSS-Syntax.

  • @charset
  • @font-face
  • @import
  • @media
  • @page
