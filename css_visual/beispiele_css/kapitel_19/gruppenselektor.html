CSS Selektoren: Gruppenselektor

Syntax:
selektorA, selektorB, selektorC

Soll die gleiche Regel für mehrere Selektoren gelten, so können diese gruppiert werden. Hierfür werden die Einzelselektoren zu einer kommagetrennten Liste zusammengefügt. Im einfachsten Fall ist es eine Liste aus Typselektoren

Beispiel:
h1, h2, h3 {font-family:arial;}

/* drei Klassenselektoren gruppiert: */
.klasseA, .klasseB, .klasseC:hover {...}

Kompatibilität:
Der Gruppenselektor wird von allen Browsern unterstützt.
