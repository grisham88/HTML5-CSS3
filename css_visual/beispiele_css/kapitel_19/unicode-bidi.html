Textproperties: unicode-bidi

Das unicode-bidi-Property bestimmt, wie bidirektionale Textinhalte zu behandeln sind. Es ist in Zusammenhang mit dem direction-Property zu betrachten.

Werte:
normal, embed, bidi-override, inherit

Beispiel:
p { text-transform: lowercase }
p { text-transform: uppercase }
