Displayproperties: display:table

Allen HTML-Elementen können seit CSS 2 über entsprechende display-Werte auch die Darstellungseigenschaften von Tabellenelementen zugeordnet werden. Hier muss darauf geachtet werden, dass mindestens die erforderlichen Rollen definiert werden. Ein Element muss daher die Rolle des <table>-Containers übernehmen (display:table), ein weiteres die Rolle der Tabellenzeile <tr>, bekommt also display:table-row. Ein Element schließlich stellt, mit display:table-cell die Tabellenzelle <td> dar.

Beispiel:
span { display:table; }

Beispiel:
<div style="display:table;width:250px; border:1px dotted; padding:5px; border-spacing:2px">
<div style="display:table-row;">
<span style="display:table-cell; border:1px solid; padding:3px;">1</span>
<span style="display:table-cell; border:1px solid; padding:3px;">2</span>
</div>
<div style="display:table-row;">
<span style="display:table-cell; border:1px solid; padding:3px;">3</span>
<span style="display:table-cell;border:1px solid; padding:3px;">4</span>
</div>
</div>
