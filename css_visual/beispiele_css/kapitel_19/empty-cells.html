Tabellen-Properties: empty-cells

Das empty-cells-Property bestimmt, ob für leere Tabellenzellen deren Rahmen, Hintergrundfarbe und -bild angezeigt oder versteckt werden soll.

Werte:
show, hide, inherit

Beispiel:
table { empty-cells:hide; }
