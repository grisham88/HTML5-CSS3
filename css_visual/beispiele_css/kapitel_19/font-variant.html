Fontproperties: font-variant

Das font-variant-Property dient mit dem Wert small-caps dazu, einen Text in Kapitälchen darzustellen, d.h. die Kleinbuchstaben durch verkleinerte Großbuchstaben zu ersetzen. Der Wert normal setzt die Darstellung auf normale Präsentation zurück, sofern die Kapitälchendarstellung von einem umgebenden Container ererbt wurde.

Werte:
normal, small-caps, inherit

Beispiel:
p { font-variant: small-caps }

Ausprobieren:
font-variant:

Sie können hierfür in den Feldern des Editierbereichs beliebige Eingaben machen, die sofort sichtbar umgesetzt werden (solange die Werte zulässig sind, selbstredend :-).
