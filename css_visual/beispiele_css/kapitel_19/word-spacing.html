Textproperties: word-spacing

Das word-spacing-Property legt den Wortabstand zwischen einzelnen Worten eines Elementinhalts fest. Der Wert normal präsentiert den Elementinhalt mit unverändertem Wortabstand, eine Längenangabe erzwingt einen bestimmten Abstand zwischen einzelnen Worten, wobei jedoch der durch das trennende Leerzeichen festgelegte Mindestabstand (offiziell) nicht unterschritten werden kann.

Werte:
normal, [laenge], inherit

Beispiel:
h1 { word-spacing: 0.5em }
p { word-spacing:0.2em; }
