Fontproperties: font-weight

Das font-weight-Property bestimmt den Fettegrad (weight) der Schrift des erfassten Containers. In der Theorie kann zwar ein abgestufter Fettegrad angegeben werden, in der Praxis liegen (bei Bildschirmausgabe) dem Browser für die meisten Schriften jedoch nur zwei alternative Schriftschnitte (normal und fett) vor.

Werte:
normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900, inherit

Ausprobieren:
font-weight:

Beispiel:
/* entspricht "400": */
p { font-weight: normal }
/* entspricht "bold": */
h1 { font-weight: 700 }
/* keine sichtbare Wirkung: */
body { font-weight: 400 }
/* wird nicht "fetter": */
strong { font-weight: bolder }

Dieser Textabsatz ist ein Beispielabsatz. Er enthält keinen sinnvollen Inhalt, sondern dient lediglich dazu, dem folgenden Absatz voranzustehen. Er wird durch die CSS-Editierung nicht erfasst. Er hat eine hellgraue Hintergrundfarbe, um besser sichtbar zu sein.

Dieser Absatz kann durch CSS-Formatierungen, die links im Editierbereich vorgenommen werden, in seiner Präsentation geändert werden. Sie können hierfür in den Feldern des Editierbereichs beliebige Eingaben machen, die sofort sichtbar umgesetzt werden (solange die Werte zulässig sind, selbstredend :-).
