Farben und Hintergründe: background-repeat

Das Property background-repeat bestimmt, ob und in welche Richtung(en) ein Hintergrundbild innerhalb der Containerfläche wiederholt (gekachelt) wird.

Werte:
repeat, no-repeat, repeat-x, repeat-y, inherit

Beispiel:
body { background-repeat: repeat-y; }
body { background-repeat: no-repeat; }
