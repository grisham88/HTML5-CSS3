Farben und Hintergründe: color

Das color-Property bestimmt die Vordergrundfarbe des Containerinhalts, mit anderen Worten die Textfarbe. Als Wert kann ein Farbname oder ein Farbwert in der für CSS üblichen Notierung eingesetzt werden.

Werte:
[farbname], [farbwert], inherit

Ausprobieren:
color:

Ausprobieren:
color:

Beispiel:
/* "rot" (Hexwert): */
h1 { color: #ff0000; }
/* "rot" (Farbname): */
span.rot { color: red }
/* "rot" (RGB-Wert): */
em { color: rgb(255,0,0) }
body { background: red; color: white; }

Dieser Textabsatz ist ein Beispielabsatz. Er enthält keinen sinnvollen Inhalt, sondern dient lediglich dazu, dem folgenden Absatz voranzustehen. Er wird durch die CSS-Editierung nicht erfasst. Er hat eine hellgraue Hintergrundfarbe, um besser sichtbar zu sein.

Dieser Absatz kann durch CSS-Formatierungen, die links im Editierbereich vorgenommen werden, in seiner Präsentation geändert werden. Sie können hierfür in den Feldern des Editierbereichs beliebige Eingaben machen, die sofort sichtbar umgesetzt werden (solange die Werte zulässig sind, selbstredend :-).
