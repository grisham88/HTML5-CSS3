Fontproperties: font-size

Das font-size-Property legt die Größe fest, in der die Schrift des Elements präsentiert werden soll. Hierbei können Größenangaben mit Schlüsselwörtern oder als Längenangabe in verschiedenen Maßeinheiten oder Prozentangaben relativ zur Schriftgröße des umliegenden Containers gemacht werden.

Ausprobieren:
font-size:

Ausprobieren:
font-size:

Werte:
xx-small, x-small, small, medium, large, x-large, xx-large. [laenge], [prozent], larger, smaller, inherit

Beispiel:
blockquote { font-size: larger }
body { font-size:smaller; }
p { font-size: 12pt; }
em { font-size: 150% }
i { font-size: 1.5em }

Die Angaben xx-small, x-small, small, medium, large, x-large, xx-large bestimmen eine absolute Schriftgröße von sehr klein bis sehr groß, die jedoch vpm Browser nachskaliert werden kann. Die Darstellungsgröße ist browserabhängig, wobei small in der Regel der HTML-Angabe size="2", die Größe xx-large der HTML-Angabe size="6"entspricht.

Die Werte smaller und bigger verkleinern bzw. vergrößern die Schrift gegen-über der Schriftgröße des umliegenden Containers um einen Faktor. Die Wirkung entspricht der HTML-Angabe size="-1" bzw. size="+1".
