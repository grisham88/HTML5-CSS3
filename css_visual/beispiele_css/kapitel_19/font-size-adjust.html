Fontproperties: font-size-adjust (CSS 2; in CSS 2.1 entfernt)

Das font-size-adjust-Property ist dazu gedacht, die x-Höhe der Schrift innerhalb des Elementcontainers an die x-Höhe der Schrift des umliegenden Containers anzupassen, sofern unterschiedliche Schriften verwendet werden. Als Ausgangspunkt und Maß für die Lesbarkeit dient der Aspektwert einer Schrift, der sich aus dem Verhältnis von Schrifthöhe zur Höhe der Kleinbuchstaben (x-Höhe) bestimmt. Besitzt eine Schrift einen kleineren Aspektwert als die Bezugsschrift, so sollte sie mittels font-size-adjust vergrößert werden, ansonsten verkleinert.

Werte:
numerisch

Ausprobieren:
font-size-adjust:

Beispiel:
Verdana (0.58) nach Comic Sans MS (0.54):
font-size-adjust: 1.07
Verdana (0.58) nach Georgia (0.50):
font-size-adjust: 1.16
Verdana (0.58) nach Times New Roman (0.46):
font-size-adjust: 1.26

Anmerkung:
Das font-size-adjust-Property wird derzeit von keinem Browser unterstützt und wurde daher aus der Spezifikation CSS 2.1 wieder entfernt.
