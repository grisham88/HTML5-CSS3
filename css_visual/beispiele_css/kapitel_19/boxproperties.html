Boxproperties

Das CSS-Boxmodell ist im Besonderen auf Blockelemente anwendbar - für Inlineelemente ist es nur eingeschränkt gültig. Aus der Sicht von CSS wird jeder Blockcontainer als "Box" betrachtet, die in vier Bereiche (Inhalt, Padding, Border, Margin) eingeteilt werden kann.

  • padding
  • margin
  • border
    • border-color
    • border-style
    • border-width
  • height
  • width
  • max-height
  • max-width
  • min-height
  • min-width
