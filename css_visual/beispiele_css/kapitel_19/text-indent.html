Textproperties: text-indent

Das text-indent-Property rückt die erste Zeile des Textinhalts eines Blockelements um den angegebenen Wert ein. Die Einrückung kann durch eine Längenangabe oder einen Prozentwert bezogen auf die Containerbreite angegeben werden. Negative Werte ("hängender Einzug") sind möglich, verursachen jedoch ohne weitere Korrektur Darstellungsprobleme (abgeschnittener Text).

Werte:
[laenge], [prozent], inherit

Beispiel:
p { text-indent: 3em }
<p style="text-indent:-2em">
Lorem ipsum ... </p>
