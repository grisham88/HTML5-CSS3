Listenstyle Properties: list-style-image

Das list-style-image-Property legt eine Grafik fest, die als Aufzählungszeichen verwendet werden soll. Defaultwert ist none (keine Grafik) – in diesem Falle tritt das list-style-type-Property in Kraft.

Werte:
url([pfad/dateiname]), none, inherit

Beispiel:
<div style="display:list-item;
list-style-image:url('pfeil.gif')">Tulpen</div>

Ausprobieren:
list-style-image:

Dieser Textabsatz ist ein Beispielabsatz. Er enthält keinen sinnvollen Inhalt, sondern dient lediglich dazu, dem folgenden Absatz voranzustehen. Er wird durch die CSS-Editierung nicht erfasst. Er hat eine hellgraue Hintergrundfarbe, um besser sichtbar zu sein.

Eine Liste:

  • Rosen
  • Tulpen
  • Nelken
