Textproperties: vertical-align

Das vertical-align-Property bestimmt die vertikale Ausrichtung eines Inlinecontainers innerhalb des durch das umliegende Blockelement zur Verfügung gestellten Raumes.

Werte:
baseline, bottom, middle, sub, super, text-bottom, text-top, top, [prozent], inherit

Beispiel:
p { line-height: 0.9;} /* Faktor */
p { line-height: 1.2em;} /* Längenangabe */
p { line-height: 140%;} /* Prozentangabe */

Ausprobieren:
vertical-align:

Test: Dieser Text kann vertikal ausgerichtet werden.
