Farben und Hintergründe: background-image

Das background-image-Property weist den Hintergrundbereich eines Elementcontainers (Textbereich, Innenabstand und Rahmenbereich) ein Hintergrundbild zu.

Werte:
url([pfad/dateiname]), none, inherit

Beispiel:
body { background-image: url('bg.gif'); }
