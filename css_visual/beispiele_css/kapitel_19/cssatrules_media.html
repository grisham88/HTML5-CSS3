CSS At-Rules: @media

Die Anweisung @media kann innerhalb eines Stylesheets als Schalter eingesetzt werden, um bestimmte, in einem Block zusammengefasste Stylesheetregeln medienabhängig auszuführen. Die Anweisung kann alternativ zu einem eigenständigen, medienspezifischen Stylesheet verwendet werden.

Beispiel:
/* Screenausgabe mit eingeblendeter Navigation: */

#nav { display:block; }
/* Druckausgabe mit ausgeblendeter Navigation: */

@media print {
#nav { display:none; }
/* weitere Änderungen für Druckausgabe: */
...
}

Kompatibilität:
Das @media-Rule wird von allen aktuellen Browsern unterstützt. Mangelhaft oder gar nicht implementiert ist es lediglich in älteren oder historischen Browsern wie IE 4 und IE 5 (Win), sowie in Netscape 4.x.
