CSS At-Rules: @charset

Die Anweisung @charset definiert den Zeichensatz des CSS-Dokuments. Dieser wird einfach als Zeichenkette (in Anführungszeichen) übergeben. Eine Bestimmung des Zeichensatzes ist für externe CSS-Dateien unter Umständen von Bedeutung. Ein Dokumentstyle wird mit dem Zeichensatz des beherbergenden HTML-Dokuments interpretiert.

Beispiel:
@charset "ISO-8859-1";

Kompatibilität:
Das @charset-Rule wird durch aktuelle Browser nicht unterstützt. In CSS 2.1 ist die Regel nicht mehr enthalten.
