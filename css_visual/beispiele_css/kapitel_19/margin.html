Boxproperties: margin

Das margin-Property legt den Außenabstand des Inhaltsbereichs eines Containers fest, englisch als "Margin" bezeichnet. Es können zwischen einem und vier Werten übergeben werden, wobei sowohl Längen- als auch Prozentangaben wie auch eine Mischung aus beidem möglich sind.

Werte:
[laenge], [prozent], inherit

Beispiel:
p { margin:0px; }
p { margin:0.2em 1.5em; }
p { margin:-1.2em 0em 0.5em; }

Ausprobieren:
margin:
...oder für jede Seite einzeln:
margin-top:
margin-right:
margin-bottom:
margin-left:
