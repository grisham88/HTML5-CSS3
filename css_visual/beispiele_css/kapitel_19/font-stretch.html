Fontproperties: font-stretch (CSS 2; in CSS 2.1 entfernt)

Das font-stretch-Property streckt oder staucht die Zeichenabstände innerhalb eines Textes.

Werte:
ultra-condensed, extra-condensed, condensed, semi-condensed, normal, semi-expanded, expanded, extra-expanded, ultra-expanded, inherit

Ausprobieren:
font-stretch:

Beispiel:
body { font-stretch:normal; }
p { font-stretch:extra-condensed; }
p { font-stretch:semi-expand; }

Anmerkung:
Das font-stretch-Property wird derzeit von keinem Browser unterstützt und wurde daher aus der Spezifikation CSS 2.1 wieder entfernt.
