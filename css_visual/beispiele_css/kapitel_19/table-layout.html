Tabellen-Properties: table-layout

Das table-layout-Property bestimmt, ob die Abmessungen der Tabelle als Höchstmaße interpretiert werden sollen, oder die Tabelle sich den Inhalten in der Breite anpassen darf. Defaultwert ist auto.

Werte:
auto, fixed, inherit

Beispiel:
table { table-layout:auto; }
