Boxproperties: border-style

Das border-style-Property bestimmt das Aussehen (style) des Rahmens. Es können, aus einer Reihe vordefinierter Schlüsselwörter, bis zu vier Werte übergeben werden. Damit das border-style-Property Wirkung zeigt, muss mindestens auch ein Wert für die Rahmenbreite border-width bestimmt sein.

Werte:
none, solid, dotted, dashed, double, groove, ridge, inset, outset, inherit

Beispiel:
p { border-style:dotted; border-width:3px;}
p { border-style:dashed; border-width:3px; }
p { border-style:double; border-width:3px; }

Ausprobieren:
border-style:
...oder für jede Seite einzeln:
border-top-style:
border-right-style:
border-bottom-style:
border-left-style:
