Farben und Hintergründe: background-position

Das background-position-Property bezieht sich auf die Position eines mit background oder background-image eingebundenen Hintergrundbildes. Das Property übernimmt Längen- bzw. Prozentangaben oder Schlüsselwörter, die einzeln oder paarweise kombiniert auftreten können.

Werte:
[prozent], [laenge], [prozent laenge], [laenge prozent], [laenge laenge],[prozent prozent], top left, top right, bottom left, bottom right, center, inherit

Beispiel:
body { background: url("bg.gif") 50px 150px }
body { background: url("bg.gif") center }
body { background-position right top }
body { background-position: 27% 3cm; }
