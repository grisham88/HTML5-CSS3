CSS Selektoren: Attributabhängige Selektoren

Auch die Attributselektoren wählen Elementcontainer innerhalb des Dokuments aus, prüfen dabei jedoch deren Attributkontext. Gewählt werden Elemente, die entweder ein bestimmtes Attribut besitzen, ohne dass dessen Wert eine Rolle spielt, Elemente deren genanntes Attribut gleichzeitig einen bestimmten Wert besitzt oder (im Rahmen einer Tokenliste) den entsprechenden Vergleichswert enthält.

Existenz eines Attributs

Syntax:
[attributname]
*[attributname]
tagname[attributname]

Ein Attributselektor gilt für alle Elemente, für die ein Attribut mit entsprechendem Namen explizit gesetzt ist (d.h. im Quelltext vorhanden ist). Die Wirkung kann auf Elemente mit bestimmtem Namen beschränkt werden.

Beispiel:
*.wichtig { border:solid 2px red; }

Attribut mit bestimmtem Wert

Syntax:
[attributname=attributwert]

Dies gilt für alle Elemente, die ein Attribut attributname besitzen, wenn dieses den Wert attributwert enthält. Der Wert wird als Zeichenkette mit Begrenzern übergeben und darf daher Leerzeichen enthalten.

Beispiel:
img[alt=""] { border:solid 2px red; }

Attribut mit Wert innerhalb einer Wertliste

Syntax:
[att~=teilwert]

Ebenso können Elemente erfasst werden, die Attribute in Form eines Listentyps (z.B. IDREFS, NMTOKENS) besitzen, sofern ein Teilwert des Attributwertes dem hier übergebenen Wert entspricht. Der übergebene Wert darf keine Leerzeichen enthalten.

Attributselektoren für Sprachcodes

Syntax:
[att|=val]

Speziell für Elemente mit zugewiesenem Sprachcode ist diese Form des Attributselektors geeignet. Hierbei muss der Attributwert mit dem übergebenen Wert beginnen, wobei und unmittelbar ein Bindestrich folgen darf.

Beispiel:
/* erfasst z.B. <p lang="EN-us">: */ [att|="EN"] { ... }

Kompatibilität:
Der Attribut-Selektor wird von allen modernen Browsern unterstützt (Mozilla, Firefox, Safari, Opera). Der Internet Explorer für Windows hingegen kennt den Attribut-Selektor erst ab Version 7.0. Der IE 5 für MacOS und der historische Netscape 4.x unterstützen diesen Selektor nicht.
