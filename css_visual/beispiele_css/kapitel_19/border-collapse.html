Tabellen-Properties: border-collapse

Das border-collapse-Property bestimmt, ob Rahmen um Tabellenzellen (sofern diese definiert sind) zusammenfallen oder, was der Vorgabe entspricht, durch Abstände separiert dargestellt werden sollen.

Werte:
separate, collapse, inherit

Beispiel:
table {border-collapse:separate}

Ausprobieren:
border-collapse:
