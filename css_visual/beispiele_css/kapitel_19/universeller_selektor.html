CSS Selektoren: *

Syntax:
*

Der universelle Selektor * (wildcard-Selektor) gilt für Elemente beliebigen Namens. Sein Einsatz macht hauptsächlich innerhalb von zusammengesetzten Selektoren oder Kontextselektoren Sinn.

Beispiel:
/* Schriftart Arial für alle Elemente: */
* { font-family:arial; }

/* In Zusammenhang mit Attributselektor : */
*[alt] {border: dotted 1px black;}

/* Platzhalter für bel. Elementbez.: */
div * {color:red}

Kompatibilität:
Der universelle Selektor * wird von allen Browsern unterstützt. Ausnahmen sind die historichen Browser Netwscape 4.x und Internet Explorer 4.0.
