CSS Selektoren: ID-Selektor

Syntax:
#IDbezeichner

Der ID-Selektor wirkt auf genau ein Element, das ein id-Attribut mit dem übergebenen Bezeichner besitzt. Er wird in der Regel verwendet, um gezielt Dokumentbereiche oder logische Elemente zu stylen, die sich im Dokument nicht wiederholen (andernfalls greift man auf Klassen zurück).

Beispiel:
p#einleitung { ... }

Kompatibilität:
Der ID-Selektor wird von allen Browsern unterstützt.
