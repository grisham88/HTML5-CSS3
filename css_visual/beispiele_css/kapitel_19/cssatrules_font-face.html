CSS At-Rules: @font-face

Mittels der Anweisung @font-face kann ein benötigter Fontsatz über deren src-Property direkt als Datei eingebunden werden. Die Spezifikation nennt hier eine Reihe verschiedener Typen, wobei man in der Praxis allerdings auf die Typen .eot (Embedded Open Type, Internet Explorer) und .prf (Portable Font Resource, Netscape 4.x) beschränkt ist.

Beispiel:
/* Import einer benötigten Schrift: */
@font-face {
src: local("Eurostyle BK");
src: url("eurostyle.eot"), url("eurostyle.pfr");
font-family: Eurostyle;
}
/* Anwendung der importierten Schrift: */
p {
font-family: Eurostyle;
font-size: 0.8em;
}

Kompatibilität:
Das @font-face-Rule wird durch aktuelle Browser mit Ausnahme des Internet Explorers nicht unterstützt, der sich dabei auf das Format EOT (embedded opentype) beschränkt. Die Verwendung ist daher als problematisch einzustufen. In CSS 2.1 ist sie nicht mehr enthalten.
