CSS Selektoren

Selektoren dienen dazu, eine CSS-Regel an ein Element oder eine Gruppe von Elementen zu binden. Sie stehen daher im Zentrum der Funktionalität von Stylesheets. Man unterscheidet einfache und zusammengesetzte Selektoren, sowie Kontext- und Pseudoselektoren.

  • Universeller Selektor
  • Typ-Selektor
  • Gruppenselektor
  • Klassenselektor
  • ID-Selektor
  • Descendant-Selektor
  • Child-Selektor
  • Adjacent-Sibling-Selektor
  • Attributabhängige Selektoren
    • Existenz eines Attributs
    • Attribut mit bestimmtem Wert
    • Attribut mit Wert innerhalb einer Wertliste
    • Attributselektoren für Sprachcodes
  • Pseudoelemente
    • :after, :before
    • :first-letter
    • :first-line
  • Pseudoklassen
    • :active
    • :focus
    • :hover
    • :lang
    • :link
    • :visited
  • Pseudo-Kontextselektoren
    • :first-child
