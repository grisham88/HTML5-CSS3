Boxproperties: border-width

Das border-width-Property bestimmt die Breite der Rahmenlinie. Es können bis zu vier Werte in Form von Längenabgaben oder dreier vordefinierter Schlüsselwörter in beliebiger Kombination übergeben werden.

Werte:
[laenge], thin, medium, thick, inherit

Beispiel:
p { border-width:2px 6px 12px;}
p { border-width:1px 24px 12px 4px; }
p { border-width:thin medium; }

Ausprobieren:
border-width:
...oder für jede Seite einzeln:
border-top-width:
border-right-width:
border-bottom-width:
border-left-width:
