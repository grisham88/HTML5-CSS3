Textproperties: direction

Das direction-Property bestimmt die Schreibrichtung innerhalb des erfassten Elements. Texte können somit auch rechts nach links dargestellt werden, wodurch es unter anderem möglich ist, Texte in Hebräisch oder Arabisch darzustellen, bei denen die umgekehrte Leserichtung gilt.

Werte:
rtl, ltr

Beispiel:
p.hebr { direction:rtl; }
p.engl { direction:lrt; }

Kompatibilität:
Das direction-Property wird derzeit von keinem Browser unterstützt.
