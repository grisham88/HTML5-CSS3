CSS Selektoren: Pseudoelemente

CSS-Pseudoelemente dienen zum Stylen von Bereichen innerhalb eines Elements oder zum Stylen von generiertem Content, zielen also, im Gegensatz zu einem Typ-Selektor oder anderen „normalen“ Selektoren nicht auf ein real existierendes Element.

:after und :before

Syntax:
selektor:after
selektor:before

Die Pseudoelement :after und :before dienen zum Einfügen von beliebigem Inhalt nach bzw. vor dem eigentlichen Inhalt des über den Selektor gewählten Elements. Der einzufügende Inhalt wird mit dem Property content angegeben und kann mit weiteren Properties beliebig gestylt werden.

Beispiel:
div#eins:before {content:url('pfeil.gif'); }

div#zwei:after {content:url('pfeil.gif'); }

div#drei:before {content:'Blindtext: '; font-weight:bold;}

div#vier:before {content: open-quote; }

div#vier:after {content: close-quote; }

:first-letter

Syntax:
selektor:first-letter

Das Pseudoelement :first-letter stylt das erste Zeichen des Inhalts des mittels des Selektors gewählten Elements. Es kann beispielsweise zur Erzeugung eines Initials verwendet werden.

Beispiel:
p { font-family:arial, sans-serif;
font-size:0.8em;
}
p:first-letter { font-size:3em;
font-weight:bold; float:left;
}

:first-line

Syntax:
selektor:first-line

Das Pseudoelement :first-line stylt die erste Zeile des Inhalts des mittels des Selektors gewählten Elements. Der Inhalt (bzw. die Länge) der Zeile hängt vom verfügbaren Platz ab, ist also unter Umständen veränderlich.

Beispiel:
p {font-family:arial, sans-serif;
font-size:0.8em;
}
p:first-line {font-weight:bold;
}
