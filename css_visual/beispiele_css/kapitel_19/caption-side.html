Tabellen-Properties: caption-side

Das caption-side-Property legt fest, ob eine Tabellenüberschrift über oder unter der Tabelle erscheinen soll. Da die Eigenschaft sich vererbt kann sie entweder direkt dem Caption-Element oder aber auch der Tabelle selbst zugewiesen werden.

Werte:
top, bottom, inherit

Beispiel:
table {caption-side:bottom;}
caption {caption-side:top;}

Ausprobieren:
