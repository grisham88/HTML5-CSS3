Farben und Hintergründe: background-attachment

Das background-attachment-Property legt fest, ob das Hintergrundbild gegenüber dem Browserfenster fixiert wird, also beim Scrollen des Dokuments stationär bleibt.

Werte:
scroll, fixed, inherit

Beispiel (s. rechts):
body {
background: url("bg.gif") no-repeat;
background-position: top center;
background-attachment: scroll;
}
