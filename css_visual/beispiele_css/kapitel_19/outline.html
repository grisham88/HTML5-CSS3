Konturproperties: outline

Das outline-Property zeichnet einen Rahmen um den Inhaltsbereich eines Containers, also innerhalb des Außenabstandes. Die Umrisslinie trägt daher, im Gegensatz zum Rahmen, wie er mit border erzeugt wird, nicht zu den Abmessungen bei. Outlines können auch Inlineelementen zugewiesen werden.

Werte:
[outline-width] [outline-style] [outline-color]

Beispiel:
p { outline: 3px solid black; }
