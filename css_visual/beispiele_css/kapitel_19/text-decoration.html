Textproperties: text-decoration

Das text-decoration-Property legt die Präsentation des Textinhalts des Containers als durchgestrichen, unter- oder überstrichen sowie blinkend fest. Der Wert none entfernt jegliche Defaultpräsentation, wird also eingesetzt, um die Unterstreichung von Hyperlinktext zu unterbinden. Es können auch, durch Leerzeichen getrennt, mehrere Präsentationsvorschriften gleichzeitig angeordnet werden.

Werte:
none, blink, underline, overline, line-through, inherit

Beispiel:
a[href] { text-decoration: underline }
<p style="text-decoration:overline underline">
Lorem...</p>
<p style="text-decoration:line-through underline"> ...</p>
