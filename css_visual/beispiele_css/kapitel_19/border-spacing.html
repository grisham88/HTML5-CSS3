Tabellen-Properties: border-spacing

Das border-spacing-Property bestimmt, gleich dem cellspacing-Attribut in HTML den Abstand zwischen den Rahmenlinien benachbarter Tabellenzellen. Es können beliebige Längenangaben vergeben werden.

Werte:
[laenge], inherit

Beispiel:
table {border-spacing:1px;}

Ausprobieren:
border-spacing:
