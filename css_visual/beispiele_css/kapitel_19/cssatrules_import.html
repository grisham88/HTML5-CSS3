CSS At-Rules: @import

Die Anweisung @import dient dem Importieren externer Stylesheetdateien. Es wird ein URL einer externen Stylesheetdatei übergeben, die relativ zum Basis-URI der aufrufenden Datei ausgewertet wird. Neben dem URL kann auch ein bevorzugter Medientyp genannt werden, für den das importierte Stylesheet zu verwenden ist. Wird kein Medientyp angegeben, so ist die Reihenfolge der Importanweisungen für den Rang der importierten Regeln relevant – die Regeln der zuletzt importierten Datei besitzen Vorrang.

Beispiel:
/* in CSS 1: */
@import url("screen.css");
@import url("print.css") print;
@import url("mobile.css") handheld;

/* in CSS 2: */
@import "screen.css";
@import "print.css" print;
@import "mobile.css" handheld;

Kompatibilität:
Das @import-Rule wird durch alle aktuellen Browser unterstützt. Der inzwischen historische Netscape 4.x kennt die Regel hingegen nicht, weshalb sie sich für diesen Browser als Stylesheetweiche einsetzen lässt.
