Boxproperties: width

Das width-Property bestimmt die Breite eines Elements. Die hier übergebene Abmessung bezieht sich, analog zum height-Property ebenfalls auf den inneren Bereich der Box. Ein Margin erhöht die Gesamtbreite entsprechend.

Werte:
[laenge], [prozent], auto, inherit

Beispiel:
p { width:200px; }
p { width:60%; }
p { width:auto; }
