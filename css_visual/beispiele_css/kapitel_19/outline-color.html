Konturproperties: outline-color

Das outline-color-Property legt die Farbe(n) der Umrisslinie oder einer ihrer Seiten fest. Es können, in Form von Farbnamen und Farbwerten in Hexadezimal- oder RGB-Schreibweise, zwischen einem und vier Werten übergeben werden.

Werte:
[farbname], [farbwert], invert, inherit

Beispiel:
p { outline-color:invert; }
p { outline-color:#555555 #999999; }
p { outline-color:red green blue; }
p { outline-color:red green blue black; }

Ausprobieren:
outline-color:
