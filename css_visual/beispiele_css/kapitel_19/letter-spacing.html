Textproperties: letter-spacing

Das letter-spacing-Property bestimmt den Abstand zwischen den Zeichen eines Textes. Der Defaultwert ist normal, was den "für den jeweiligen Schrifttyp üblichen" Wert bedeutet. Eine Längenangabe definiert alternativ den gewünschten Zeichenabstand, wobei ein positiver Wert den Abstand vergrößert, ein negativer diesen verringert.

Werte:
normal, inherit, [laenge]

Beispiel:
blockquote { letter-spacing: 0.1em }
p { letter-spacing: 2px }

Ausprobieren:
letter-spacing:

Kompatibilität:
Das letter-spacing-Property wird von allen aktuellen Browsern unterstützt. Lediglich der historische Netscape 4.x kennt dieses Property nicht.
