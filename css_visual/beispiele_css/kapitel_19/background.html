Farben und Hintergründe: background

Das Property background dient als Sammelproperty für die Bestimmung der Eigenschaften des Hintergrunds eines Containers. Mittels background können Hintergrundfarbe (background-color), Hintergrundbild (background-image) und dessen Eigenschaften festgelegt werden. In diesem Zusammenhang kann angegeben werden, ob das Hintergrundbild gegenüber dem Browserfenster fixiert werden soll (background-attachment), ob und in welchen Richtungen es gekachelt werden soll (background-repeat) und wo es positioniert wird (background-position).

Werte:
[background-color] [background-image] [background-attachment] [background-repeat] [background-position]

Beispiel:
/* Hintergrundfarbe rot: */
p { background: red }
/* Hintergrundfarbe teilw. sichtbar: */
p { background:#ff9999 url('bg.gif') repeat-y;}
/* posit. Hintergrundbild; s. re: */ p { background:url('bg.gif') 100px 25px no-repeat;}
